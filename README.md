## 📖 Description

Application web complète permettant de gérer une liste d'utilisateurs avec les opérations **CRUD** (Create, Read, Update, Delete).

- **Frontend** : React.js avec une interface moderne et responsive
- **Backend** : API REST avec Express.js
- **Communication** : Axios pour les requêtes HTTP

---

## ✅ Fonctionnalités

| Opération | Description | Statut |
|-----------|-------------|--------|
| **Create** | Ajouter un nouvel utilisateur | ✅ |
| **Read** | Afficher la liste des utilisateurs | ✅ |
| **Update** | Modifier un utilisateur existant | ✅ |
| **Delete** | Supprimer un utilisateur | ✅ |

### Fonctionnalités supplémentaires

- 🎨 Interface moderne avec dégradés et animations
- 📱 Design responsive (mobile, tablette, desktop)
- 👤 Avatars générés automatiquement
- ⏳ Indicateur de chargement
- 💬 Confirmation avant suppression
- 🎯 Messages de succès
